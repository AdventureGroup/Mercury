# Mercury 水银
## Summary 概要
这是一个使用Unity制作的2D横板RPG向小游戏
## Structure 结构
* `Assets\Scenes` 场景目录
* `Assets\Scripts` 脚本目录
  * `ExternalLibrary` 外部库源码目录
* `Assets\Setting` 游戏配置目录
* `Assets\Prefabs` 预制体目录
* `Assets\Resources` 资源目录，但是只用于存放配置表，不存放任何资源
* `Assets\Textures` 材质目录
## Tags & Layers
* Tags
  * `Entity` 实体
* Layers
  * `ConstPlatform` 不可穿越的平台
  * `OneWayPlatform` 可穿越的平台
